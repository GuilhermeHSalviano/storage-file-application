
# Project Title

This project is a simple web API which I used to learn how to storage and recover files physically. At the moment, I haven't used a relational database to store the name of the file because my main concern was to be able to store files. Maybe later I will dedicate my time to create a hole application that implements relational database beside file physical storage.

## Lessons Learned

With this project I have learned about the IFileForm interface, Globally Unique Identifier method and other resources I've had to use to implement the whole API. I know there is still a lot of stuff to learn, but I was really curious about how an API can storage files and I got very impressed with how easy it is at first glance.
